수평 정렬(Horizontal Align)

Inline 요소를 정렬하고 싶을 때

parent에 text-align: center

자식

Block 요소를 정렬하고 싶을 때

정렬하고 싶은 block 요소에 width를 정해준 뒤, margin: auto 설정

자식

여러 개의 Block 요소를 정렬하고 싶을 때

1. 정렬하고 싶은 block 요소에 동일하게 width를 정해준 뒤, margin:auto 설정
2. 정렬하고 싶은 block 요소를 inline-block 등 inline 요소로 변경한 뒤, 부모에게 text-align: center 설정

수직 정렬(Vertical Align)

Inline 요소를 정렬하고 싶을 때 (Block 요소도 가능)

부모에 위 아래 padding을 동일하게 줘서 부모와 자식 사이에 간격을 띄워서 정렬된 것처럼 보이게 만듦

자식

Inline 여러 줄 요소를 정렬하고 싶을 때

위에 있는 padding 방법으로도 가능하고, table 형태로 구성해서 vertical: middle을 줌

Block 요소를 정렬하고 싶을 때

부모 요소에 position: relative(static이 아니면 O)를 주고, 자식에게 position: absolute를 줘서 부모 기준으로 이동하게 만듦 left: 50%, transform: translateX(-50%)로 수평 정렬 top: 50%, transform: translateY(-50%)로 수직 정렬

자식

Inline, Block 요소 모두 가능하면서 가장 간단한 방법

부모를 grid 혹은 flex로 만든 후, 부모에게 아래 내용을 적용 justify-content: center 수평 정렬 align-items: center 수직 정렬
